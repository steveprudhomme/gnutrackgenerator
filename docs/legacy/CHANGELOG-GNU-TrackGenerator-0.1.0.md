## [0.1.0] - 2026-07-01

### Added

- Initial CustomTkinter desktop interface.
- Dynamic rows for programmable musical segments.
- BPM, numerator, denominator and measure count fields.
- Support for asymmetric and complex meters such as `7/8` and `27/16`.
- Project save/load support through `.gen` JSON files.
- LilyPond `.ly` generation using `DrumStaff` and `\drummode`.
- MIDI generation through LilyPond subprocess calls.
- WAV rendering through FluidSynth or TiMidity subprocess calls.
- Basic validation and user-facing error handling.
- Initial project documentation: `README.md`, `ROADMAP.md`, `CHANGELOG.md`.
