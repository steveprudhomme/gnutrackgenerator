# Contributeurs et contributrices

Ce fichier reconnaît les personnes et formes de contribution qui font avancer GNU TrackGenerator.

Le projet valorise autant les contributions techniques que non techniques. Une amélioration de documentation, un test de reproduction, une traduction, une correction d’accessibilité ou un tri de ticket peut être aussi importante qu’un changement de code.

## Principes de reconnaissance

- Les contributions doivent être reconnues publiquement lorsque la personne le souhaite.
- Les contributions individuelles demeurent attribuées dans l’historique Git.
- Les commits doivent être signés selon le DCO avec `Signed-off-by`.
- Une personne peut demander à ne pas être listée dans ce fichier.
- Les reconnaissances ne créent pas automatiquement des droits d’écriture ou un rôle de maintenance.

Format recommandé :

```text

```

Ne pas ajouter d’adresse courriel privée sans consentement explicite.
